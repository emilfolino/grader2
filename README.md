# Grader 2

### Setup

Create a file `.env.local` and add your token and the id of the Canvas course to it.

```env
COURSE_ID=59505
CANVAS_TOKEN="123123..."
```



### Usage

Start the grader and see its menu.

```bash
npm start
```



### Development

Run the linter and the fixer.

```bash
npm run eslint
npm run eslint:fix
```
